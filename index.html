How to use
  • Save this file as index.html in your repo (e.g. curtelbow.github.io or curtelbow.com source).
  • Commit + push, then open your GitHub Pages URL in the browser to preview.
  • Update links for “Listen” and “Watch” as you publish real content.
Quick edits
  • Change the big headline in the <h1 class="headline"> tag.
  • Swap the tagline paragraph under <p class="tagline"> with new copy.
  • Adjust animation timing by tweaking the --delay values inline.
